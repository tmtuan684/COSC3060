Breakpoints

Breakpoints are predefined width before which element's width is 100% and after which the element's width is a predefined max-width.

.container sets max-width at each responsive breakpoint
100% wide until small breakpoint where screen size < 576px (predefined width=540px) (phone)
100% wide until medium breakpoint where screen size < 768px (predefined width=720px) (tablet)
100% wide until large breakpoint where screen size < 992px (predefined width=960px) (laptop)
100% wide until extra large breakpoint where screen size < 1200px (predefined with=1140px) (FHD monitor)
100% wide until extra extra large breakpoint where screen size < 1400px (predefined with=1320px) (UHD monitor)
.container-fluid sets width to 100% at all breakpoints
